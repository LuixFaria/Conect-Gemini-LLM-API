# Conect-Gemini-LLM-API
Cod de estudo - Luiz Guilherme Faria Ferreira

Este projeto é um exemplo simples de integração com a API Google Generative AI (Gemini) usando Python.

- O script lê a chave de acesso a partir de um arquivo externo
- Lista os modelos disponíveis
- Envia um prompt para o modelo selecionado
- Exibe a resposta no terminal

O objetivo é demonstrar de forma prática como configurar a biblioteca, realizar uma geração de conteúdoe preparar o ambiente para futuras aplicações com IA generativa.
